Grid

.large-[#] .medium-[#] .small-[#] .column
Basic Grid

The grid is really easy to use. for instance if you put .large-4.columns on a div it will span 4/12ths of it's parent container on "Large" screens. When resized below "large" it will break down to 100% width. This can be any combination of numbers as long as it ads up to the number of your total columns.

Targeting Screen Sizes

Often times you need medium and small screens to display multiple columns. You can easily target those screens using the .small or .medium column class. By default, these sizes cascade. so if you put small-4 on a div. It will render as 4/12ths width on small, medium and large.

Offsets

If you need some space between your columns you can use the offset class. Below we declare the column width (large-5) and the offset for the large screen (large-offset-1). Since we only declared large a large offset it will only apply to large screen widths. it breaks down to 100% width on medium and small.
